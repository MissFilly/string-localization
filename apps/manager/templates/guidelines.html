Guidelines

Please read carefully this guide before strarting to translate.

Here is a list a things you need to know and pay attention to while translating.

  • 1. What are these words about?

    All sentences you will translate are part of the mobile applications we are developing.

    If you want to know what our apps are about and what they do, you can visit our website or see our store.

  • 2. Uppercase and lowercase letters

    It is important that you respect uppercase and lowercase in the original sentence (the sentence in English) at all times. For example:

    Hello, world would be translated in Spanish as Hola, mundo.

    Hello, World would be translated as Hola, Mundo.

  • 3. Punctuation

    The same way, you must follow the same punctuation as the original string.

    Hello, world. would be translated in Spanish as Hola, mundo..

    Hello, world would be translated as Hola, mundo.

    Hello, world! would be translated as ¡Hola, mundo!.

  • 4. String formatting

    Some sentences contain symbols that later our apps will replace with another letter, sentence or number. You'll normally encounter these symbols as %s or %@.

    They act like wildcards, so a sentence like Hello, %s!, after our app replaces the symbol with the corresponding name, could be Hello, John!, Hello, Mary!, Hello, beautiful person!, etc.

    Knowing this, you should follow the pattern so the sentence makes sense in the translated language. For example:

    Does %s like apples? would be translated in Spanish as ¿A %s le gustan las manzanas?.

  • 5. String description

    Some strings have a description of the context they are used in, so you can choose the most appropriate translation.

  • 6. Other symbols

    You may also encounter series of symbol in sentences, like &#8230; in Processing&#8230;. These codes replace special characters in the sentence.

    To see what each symbol means, you can check a list of HTML codes (in this case, &#8230; is the code for the horizontal ellipsis character, …), but you must respect them in your translation, i.e.:

    Processing&#8230; would be translated in Spanish as Procesando&#8230;.

  • 7. Typos and non-translatable strings

    It is possible that you encounter errors in the original sentences. These could be typographical or even ortographical. We would be very thankful if you let us know in case you find one of these!

    Also, there are some strings in our database that are not supposed to be translated, for example our apps' names. If you find a sentence that doesn't look at all like it should be shown to our users or translated, please let us know too!
