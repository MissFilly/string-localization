Guideline

Please read carefully this guide before strarting to translate.

Here is a list a things you need to know and pay attention to while translating.

  • Uppercase and lowercase letters

    It is important that you respect uppercase and lowercase in the original sentence (the sentence in English) at all times. For example:

    Hello, world would be translated in Spanish as Hola, mundo.

    Hello, World would be translated as Hola, Mundo.

  • Punctuation

    The same way, you must follow the same punctuation as the original string.

    Hello, world. would be translated in Spanish as Hola, mundo..

    Hello, world would be translated as Hola, mundo.

    Hello, world! would be translated as ¡Hola, mundo!.

  • String formatting

    Some sentences contain symbols that later the program of our app will replace with another letter, sentence or number. You'll normally encounter these symbols as %s or %@.

    They act like wildcards, so a sentence like Hello, %s!, after our app replaces the symbol with the corresponding name, could be Hello, John!, Hello, Mary!, Hello, dog!, etc.

    Knowing this, you should follow the pattern so the sentence makes sense in the translated language. For example:

    Hello, %s! would be translated in Spanish as ¡Hola, %s!.

    It is possible that at some point of time you encounter errors in the original sentences. These could be typographical or even ortographical. We'd be very thankful if you let us know in case you find one of these!

    Also, there are some strings in our database that are not supposed to be translated, for example our apps' names. If you find a sentence that doesn't look at all like it should be shown to our users or translated, please let us know too!
